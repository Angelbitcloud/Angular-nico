Crear, eliminar y mostrar son operaciones fundamentales en el contexto de las bases de datos.

Crear: Operación que consiste en generar una nueva entidad o estructura en la base de datos, estableciendo su estructura y restricciones.

Eliminar: Operación para eliminar registros o documentos existentes en la base de datos que ya no son necesarios.

Mostrar: Operación para recuperar y visualizar datos almacenados en la base de datos, obteniendo información específica según criterios de búsqueda.
