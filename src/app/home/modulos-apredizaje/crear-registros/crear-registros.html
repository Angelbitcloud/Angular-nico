Insert Into: Es una sentencia SQL utilizada para insertar nuevos registros en una tabla. Permite especificar los valores a insertar para cada columna de la tabla.

Select: Es una sentencia SQL utilizada para obtener registros de una tabla o conjunto de tablas. Permite especificar las columnas que se desean recuperar, las condiciones de filtrado y el orden de los resultados.

Where: Es una cláusula utilizada en la sentencia SQL Select (y otras) para filtrar los registros que cumplen una determinada condición. Permite establecer condiciones lógicas y comparativas para restringir los resultados.

Delete: Es una sentencia SQL utilizada para eliminar registros de una tabla. Permite especificar las condiciones que deben cumplir los registros a eliminar.

Update: Es una sentencia SQL utilizada para actualizar los valores de uno o más registros en una tabla. Permite modificar los valores de las columnas existentes en función de ciertas condiciones.
