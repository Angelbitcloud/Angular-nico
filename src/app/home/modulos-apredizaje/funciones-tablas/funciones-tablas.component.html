Columnas Calculadas: Son columnas que se definen en una tabla pero cuyos valores se calculan automáticamente en función de otras columnas. Estas columnas se pueden crear utilizando expresiones y funciones en SQL para realizar cálculos o combinaciones de valores de otras columnas.

Funciones Matemáticas: Las funciones matemáticas en SQL son operaciones que se aplican a valores numéricos para realizar cálculos matemáticos. Esto incluye funciones como SUM (suma), AVG (promedio), MAX (máximo), MIN (mínimo) y COUNT (conteo), entre otras. Estas funciones son útiles para realizar operaciones matemáticas en conjuntos de datos almacenados en tablas.

Clausula de Orden: La cláusula ORDER BY se utiliza en SQL para especificar el orden en el que se deben devolver los registros de una consulta. Permite ordenar los resultados de una consulta en función de una o varias columnas, ya sea de forma ascendente (ASC) o descendente (DESC).

Operaciones Lógicas: Las operaciones lógicas en SQL permiten realizar evaluaciones basadas en condiciones lógicas. Esto incluye operadores como AND, OR y NOT, que se utilizan para combinar condiciones y realizar evaluaciones booleanas en las consultas. Estas operaciones son útiles para filtrar datos en función de condiciones específicas.

Patrones de Búsqueda: SQL proporciona operadores y funciones para realizar búsquedas de patrones en cadenas de texto. Esto incluye el uso del operador LIKE, que permite buscar registros que coincidan con un patrón específico utilizando caracteres comodín como '%' (cualquier cadena de caracteres) y '_' (cualquier carácter único).

Funciones de Agrupamiento: Las funciones de agrupamiento en SQL se utilizan junto con la cláusula GROUP BY para realizar cálculos en grupos de registros en lugar de en el conjunto completo de datos. Estas funciones, como SUM, AVG, MAX, MIN y COUNT, se aplican a conjuntos de registros que cumplen con ciertas condiciones y devuelven resultados agregados, como la suma total, el promedio, el máximo valor, el mínimo valor y el conteo de registros en cada grupo.
