Primary Key

Foreign Key

Primary Key (Clave Primaria): La Primary Key es un atributo o conjunto de atributos que identifica de forma única cada registro en una tabla. Sirve como identificador único para cada fila y garantiza la unicidad y la integridad de los datos en la tabla. Cada tabla puede tener solo una Primary Key y no puede contener valores duplicados ni valores nulos.
Foreign Key (Clave Externa): La Foreign Key es un atributo o conjunto de atributos en una tabla que establece una relación con la Primary Key de otra tabla. Sirve para mantener la integridad referencial y establecer una conexión entre dos tablas relacionadas. La Foreign Key en una tabla se refiere a la Primary Key correspondiente en otra tabla y puede tener valores que coincidan con los valores de la Primary Key o ser nulos si la relación es opcional.
